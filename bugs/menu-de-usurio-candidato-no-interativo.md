## Bug: Menu de usuário "Candidato" não é interativo

### Severidade
Medium

### Descrição
O elemento de menu localizado no canto superior direito com o rótulo "Candidato" não responde a interações de clique

### Passos para reprodução
1. Realizar login na aplicação
2. Localizar o menu "Candidato" no canto superior direito
3. Clicar no elemento

### Resultado esperado
Exibição de menu dropdown com opções (ex: perfil, logout, etc.)

### Resultado atual
Nenhuma ação é executada ao clicar

### Impacto
Usuário não consegue acessar funcionalidades relacionadas à conta

### Ambiente
- Browser: Chrome
- Resolução: Qualquer
